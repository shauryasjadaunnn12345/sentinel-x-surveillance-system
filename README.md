# 🛰️ SENTINEL-X

### AI-Based Multi-Node Defence Surveillance System

---

## 🚀 Overview

SENTINEL-X is a low-cost, AI-powered surveillance system designed to detect, classify, and evaluate threats in real-time.

It integrates **computer vision, simulated sensor inputs, and intelligent threat scoring** to reduce false alarms and improve decision-making in security-critical environments.

---

## 🎯 Problem Statement

Traditional surveillance systems:

* Cannot differentiate between types of threats
* Generate high false alarms
* Are expensive and not scalable

---

## 💡 Solution

SENTINEL-X introduces:

* Multi-node surveillance architecture
* AI-based object detection
* Threat classification (Human, Drone, Animal)
* Dynamic threat scoring system

---

## ⚙️ How It Works

1. Motion is detected (simulated sensor input)
2. Signal triggers AI processing
3. Object is detected using computer vision
4. Threat is classified
5. Threat score is calculated
6. Alert is generated
